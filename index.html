Desafio DIO Sobre as Tags Ensinadas

h1, h2, h3, h4, h5 e h6

São os 'headings', comportando-se como títulos. O número indica qual nível o título é, além de, quanto maior o número, menor o tamanho da letra.

p

Utilizada para criar um parágrafo

mark

Utilizado para destacar, como se fosse um marca texto, um trecho do texto ou uma palavra.

small, i, u, strong

Tais tags conferem uma característica ao texto de, respectivamente, deixar o texto menor, deixa-lo em itálico, colocar uma underline e colocar negrito.

ol, ul, li

São tags para se criar uma lista, sendo a 'ol' lista ordenada e a 'ul' lista não ordenada. A li será utilizada para adicionar os componentes da lista.

hr

Serve para colocar uma linha divisória na página.

blockquote

Serve para se fazer uma citação.

a

É tipo uma âncora, servindo, normalmente, para te levar a um outro lugar no site ou, também, a outros sites.

sub e sup

Servem, respectivamente, para deixar uma palavra ou trecho mais baixo ou mais elevado.

font

Uma tag que não é mais suportada no HTML5, servia para colocar a fonte(tamnho, cor...) do texto.

del

Serve para deixar um traço horizontal no meio da palavra ou trecho do texto.

abbr

Serve quando você quer escrever a abreviação de uma palavra e, qnd você passa o mouse emcima, aparece a palavra completa.
